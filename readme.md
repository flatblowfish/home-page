### 描述（Description）

一个简单的主页生成器。A simple home page generator.

### 使用

1.安装依赖。

```shell
npm install
```

2.编辑根目录和主题目录的`config.yml`。

3.生成静态网页。

```shell
gulp
```

4.部署`dist`文件夹中的内容到网站目录。
